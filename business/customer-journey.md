# Customer Journey Map for Finance-Sentry
## Aware Phase
### Trigger Event
- Financial service company experiences a transaction processing issue
- Operations team receives a complaint from a customer about a transaction issue
- Company's existing transaction monitoring system fails to detect an issue

### Friction Points
- Lack of visibility into transaction processing
- Inefficient manual investigation process
- Limited automation for issue resolution

### User Emotions
- Frustration
- Anxiety
- Concern

### Opportunities to Delight
- Real-time transaction tracking
- Automated issue detection and resolution
- Proactive alerts for potential issues

- Awareness Metric: Number of companies experiencing transaction processing issues per quarter

- Operations team recognizes the need for a more efficient transaction monitoring system

### User Emotions

- Frustration
- Uncertainty
